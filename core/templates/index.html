About Me

If you're here, you probably clicked on a link to this site I sent to you. I'm SharpBit, the creator of this website. My two favorite hobbies are playing games and coding. My favorite games are Rocket League, Minecraft, and Town of Salem. I've also been playing violin since I was 5 and I currently play in the MYO Symphony Orchestra. I've participated in All County in grades 5, 6, 8, and 9 and LISFA in grade 10.

I first started out with Scratch in 6th grade where I made games and programs and competed against my friend, Sayajiaji. However, I started to learn Python in August 2017, which is when I realized programming is really fun and not difficult! I also know some Javascript, Go, Java and C++. I created this website to learn more about web development. You can view the source code here. My most used project is brawlstats, an API wrapper written in Python for the Brawl Stars API. You can view all of my projects on my github page. If you want to learn python too, I compiled a list of useful resources to learn Python!
